# 最近更新：

[2019-04-27]



在2019-04-26的早上和群友聊了一下，

群友反馈说：“状态栏震动提示图标和周围风格不搭。”、

主要就是，除状态栏图标外，其它图标为扁平风格。

就它一个为线条感。

然后我就有了要不自己画一下？

作为一个计算机学生就要就要体现出，自己的动手能力，然而我是个手残 >_<~~~。

就去”借”了一个ipad开始就开始了。。。。

主要工具：ipad + Apple Pencil + 有空闲时间的脑 + 手

### 第二个图标:状态栏环形电量提示图标

怎么说了.....

虽然我不用这个，我是直接显示百分比，但是这个和目前的振动图标情况是一样的！但是它的情况会好很多。

以下为我的一个想法...

想着挺好的,但是画出来效果是真的不好,可能是手残.....

主要的想法就是,用黑色部分显示电量的百分比,

以前为一个环,看着有点空洞.

关于普通充电时的动画:可以用一个圆充电池的中心逐渐扩大到外围,用外围圆的厚度表示已充电量.
