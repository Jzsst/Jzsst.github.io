一、C语言

就本人而言不是很喜欢VC6.0，为啥？

1、VC6.0安装麻烦，全是玄学。

2、详细介绍，软件和编译环境版本过老。

3、丑

优点：

1、免费（虽然其他软件也免费）

2、上手简单，网上使用教程多。

3、考试通用。

个人推荐：

1、Dev-C++：简单、小巧、汉化、操作简单。

2、codeblocks：简单、适合项目开发、易汉化、兼容好。

3、MingW + Clion：jetbrains公司软件生态、功能强大。

我的选择 ：MingW + Clion

环境配置

1、2、种推荐直接下载软件安装即可。

VC6.0建议使用免安装版，如果是安装版，安装一步步来就好了，希望不要出错！

重点：MingW + Clion的配置

1、下载MingW[下载链接]

2、解压exe文件，得到MingW文件。

3、将MingW文件夹放置在一个合适的位置。如：其它软件安装位置。

4、将WingW文件夹下的bin加入环境变量Path中即可，注意加入时格式的书写。

5、检查，如图为正常

6、Clion导入MingW即可

3、将jdk****文件夹下面的bin文件加入到环境变量Path中即可。

4、检查，如图即可。

四、Android

其实这里我就优点偷巧了！！！

1、直接通过android studio进行管理SDK和SDK tools

2、选择自己需要的，然后点击Apply，就会提示你下载。

3、配置androidSDK环境变量

五、Flutter

Flutter只要基于androidSDK修改添加一下就好了。

参考文献

1、添加两个用户变量

PUB_HOSTED_URL | https://pub.flutter-io.cn

FLUTTER_STORAGE_BASE_URL | https://storage.flutter-io.cn

2、下载Flutter其最新可用的安装包，点击下载 。

3、将安装包zip解压到你想安装Flutter SDK的路径

（如：E:\flutter；注意，不要将flutter安装到需要一些高权限的路径如C:\Program Files\）

4、在Flutter安装目录的flutter文件下找到flutter_console.bat，

双击运行并启动flutter命令行，接下来，你就可以在Flutter命令行运行flutter命令了。

5、添加Flutter环境变量和androidSDK环境变量

6、android studio 或者 IntelliJ IDEA 配置

安装插件Flutter和Dart插件

7、检查，输入命令：flutter doctor
